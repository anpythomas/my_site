Booggle



I collect rare whiskey. Unfortunately, it's difficult to find secondary pricing data. I built Booggle to help fellow enthusiasts value their collections. This tool scrapes whiskey sales from across the web and plots the data on easy to read charts. Built with Django, hosted on AWS using Postgres DB Engine, this site is scalable and currently growing. Still a work in progress... Cheers 🥃

The source code for this project is on a private Github repo. Please contact me for a redacted version.

Real Estate Prices



An object-oriented programing (OOP) tool used by professional real estate developers to assist with pricing. This tool scrapes requested information from an online resource and auto-uploads data to the clients Google Forms account. (The source code is for an older iteration of the program. The current version removes outliers (view algo source code) , allows filtering of land size, provides the client with price per acre and the average price of similar lots in the area.)
